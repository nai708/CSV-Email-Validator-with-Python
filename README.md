# CSV Email Validator with Python

## Objective

The goal of this project was to practice Python programming while applying quality assurance concepts such as:
- Data validation
- Defect identification
- Input verification
- Automated testing logic
- Reporting validation results

## Features
- Reads mock user data from a CSV file
- Validates email addresses using regular expressions
- Identifies invalid email formats
- Displays row numbers containing invalid records
- Generates a validation summary
- Counts valid and invalid email addresses

## Example Output

FAIL: row 27 has an incorrect email format. Invalid email: exampleemail.com

Validation Summary:
Total number of invalid email addresses: 1
Total number of valid email addresses: 999
Total rows processed: 1000

## Technologies Used
- Python
- CSV file
- Regular Expressions (re)
